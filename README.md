# Vite + Bun + Three.js starter

Minimal starter template to quickly bootstrap **Vite**, **Bun**, and **Three.js** (TypeScript, ESM).

## Requirements

- [Bun](https://bun.sh/) installed

## Getting started

Install dependencies:

```bash
bun install
```

Development (HMR):

```bash
bun run dev
```

Build:

```bash
bun run build
```

Preview the production build:

```bash
bun run preview
```

## Structure

- `src/main.ts`: entry point (mounts the app into `#app`)
- `three`: main dependency (`three` in `package.json`)
